Ubuntu GUI on Windows 10

18 December 2018 on ubuntu, tutorials

I got it working by installing x11-apps to get the X subsystem, then running a X listener like XMING or XManager on the Windows10 side.

export DISPLAY=:0

(You also can add export DISPLAY=:0 to the last line of your user's .bashrc file)

Then run xclock to verify.
