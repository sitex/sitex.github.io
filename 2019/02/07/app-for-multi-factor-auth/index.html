App for multi-factor auth

07 February 2019 on scripts

A simple application for multi-factor authentication, written in HTML using jQuery Mobile, jsSHA, LocalStorage and Application Cache.

It implements the TOTP (Time-Based One-Time Password) algorithm according to RFC6238 and has been tested to work with Google Authenticator, Dropbox, Dreamhost, Amazon, Linode, Okta and many other services.

https://github.com/gbraad/gauth/
