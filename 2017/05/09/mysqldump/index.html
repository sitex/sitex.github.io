mysqldump

09 May 2017 on mysql

Simple usage of mysqldump:

mysqldump --user='user' --password='password' --skip-lock-tables db_name table_name | gzip -c > /home/hosting/table_name.sql.gz

Use --skip-lock-tables.
Note that % should be \%.
